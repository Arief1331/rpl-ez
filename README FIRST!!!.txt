Hal-hal yg perlu diperhatikan saat jalankan aplikasi ini di LocalHost (XAMPP, MAMP, LAMP) :
1. Pergi ke direktori database, dan open file ezeats.sql
2. jalankan semua baris kode ezeats.sql di SQL Shell (postgresql)
3. pergi ke direktori php/functions.php dan open file dan ganti $user
dan $password sesuai dg punya kalian (sesuai dg username dan password di SQL Shell)
4. Finally, aplikasi sudah bisa digunakan.
